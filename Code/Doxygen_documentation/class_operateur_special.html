Detailed Description

Classe abstraite représentant les opérateurs spéciaux comme CLEAR, UNDO et REDO.

Ces méthodes ne dépilent ni n'empilent d'opérandes, mais les méthodes sont présentes pour mettre en exergue le polymorphisme. En effet, les opérateurs spéciaux sont bien des opérateurs.
