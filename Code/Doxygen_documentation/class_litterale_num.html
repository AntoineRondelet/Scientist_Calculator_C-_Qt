Detailed Description

Classe abstraite représentant les littérales numériques, à savoir les littérales réelles, entières et rationnelles.

La classe gère la création de littérales numériques. Elle est dotée de l'opérateur d'affectation = , et peut aussi transformer la LitteraleNum de la classe en littérale complexe dont la partie imaginaire serait nulle.
